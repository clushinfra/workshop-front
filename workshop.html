CI/CD 구축 실습 워크숍

Git → Jenkins → Nexus → ArgoCD

이 워크숍은 GitOps 기반의 CI/CD 파이프라인을 실습하며 Kubernetes 환경에서 실제 배포 자동화 흐름을 체험해보는 시간입니다.

1. GitOps 기반 CI/CD 파이프라인 구축

  • GitLab 저장소를 Jenkins와 Webhook으로 연동
  • Jenkins Job에서 Dockerfile을 기반으로 이미지 빌드
  • Nexus Registry에 Docker 이미지 Push
  • ArgoCD가 Git 상태 변경 감지 → Kubernetes에 자동 반영
Git → Jenkins → Nexus → ArgoCD → Kubernetes

2. 클러스터 모니터링 환경 구축

  • Helm Chart를 사용한 Prometheus + Grafana 설치
  • 모니터링 항목: CPU, Memory, Pod 상태, 네트워크 등
  • Grafana Dashboard를 통한 실시간 시각화
